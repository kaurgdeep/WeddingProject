# Wedding Planner
Creating a web app that allows registered users to plan a wedding. Visitors can RSVP to weddings and view attendee lists.

**Key Features** 
------------------
#### • Register • LogIn • Create wedding and venue • RSVP • Un-RSVP 
#### • Delete a wedding • Get details
----------------
